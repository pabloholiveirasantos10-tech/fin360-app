Segurança
Criptografia de
nível bancário.
O mesmo padrão usado por bancos e governos — implementado diretamente no navegador, sem depender de ninguém.
AES-256-GCM
Cifra simétrica de 256 bits com autenticação de mensagem integrada. Impossível de quebrar por força bruta com os computadores atuais.
PBKDF2 · 100.000 iterações
Sua senha passa por 100 mil rodadas de hashing com SHA-256 e salt aleatório. Ataques de dicionário se tornam computacionalmente inviáveis.
100% local · Web Crypto API
Toda criptografia ocorre no navegador usando a API nativa do sistema operacional — nenhum dado trafega pela rede em nenhum momento.
Zero conhecimento
Nem nós sabemos sua senha. Não existe servidor, banco de dados ou log. Se você perder a senha, ninguém no mundo consegue recuperar os dados.

FAQ
Perguntas
frequentes.
Meus dados ficam seguros?
Sim. Tudo é criptografado com AES-256-GCM e armazenado localmente no seu navegador. Nenhum dado é enviado para servidores — nem os nossos.
Preciso de internet para usar?
Não. Após o primeiro carregamento o Fin360 funciona 100% offline. Internet só é necessária para carregar fontes e ícones na primeira abertura.
Posso usar em vários dispositivos?
Sim. Exporte o CSV como backup e importe no outro dispositivo. O arquivo HTML roda em qualquer navegador moderno — Chrome, Firefox, Edge, Safari.
O que acontece se eu esquecer a senha?
Por design de segurança, a senha não pode ser recuperada — isso garante que ninguém além de você acesse seus dados. Mantenha backups CSV regulares.
Tem algum custo?
Zero. O Fin360 é gratuito sem planos pagos, sem anúncios, sem coleta de dados. Um arquivo HTML que você baixa uma vez e usa para sempre.
O cofre de senhas é realmente seguro?
Sim. Usa AES-256-GCM (o padrão ouro da criptografia) com chave derivada via PBKDF2 com 100.000 iterações. Cada sessão usa um IV único gerado aleatoriamente.
